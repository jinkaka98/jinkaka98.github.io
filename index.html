FEATURES

MULTI-PROVIDER

Connect OpenAI, enowX, Trae, and more. Route requests to any provider from a single endpoint. Priority-based routing rules.

IDE COMPATIBLE

Works with VS Code, Cursor, Windsurf, and any tool that supports OpenAI API format. Zero configuration needed.

LICENSE MANAGED

Secure license activation with offline caching. Admin controls access. 7-day offline grace period with auto re-validation.

STREAMING

Full SSE streaming support. Real-time token-by-token responses. Transparent proxy with zero latency overhead.

ENCRYPTED

API keys encrypted at rest with AES-256-GCM. Machine-bound key derivation. Your credentials never leave your device.

AUTO-UPDATE

Built-in auto-update from GitHub Releases. SHA256 checksum verification. Automatic rollback on failure.
